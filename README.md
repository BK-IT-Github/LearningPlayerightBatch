# Learning Playwright Batch

A small Playwright learning repository with example JavaScript files in `chapter_01_Basics`.

## Project Description

This repository contains basic Playwright examples and notes for learning browser automation with Playwright.

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Run Playwright tests (if added):

```bash
npx playwright test
```
